- **Hierarchical Clustering**: Builds a hierarchy of clusters using agglomerative approach
- **DBSCAN**: Density-based clustering that identifies clusters of varying shapes
- **GMM**: Probabilistic model that assumes data is generated from a mixture of Gaussian distributions
